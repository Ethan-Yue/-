# 在线简历部署指南

## 本地预览

双击 `index.html` 即可在浏览器中打开，或在本目录运行：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

然后访问 http://localhost:8080

---

## 推荐托管平台（免费）

### 1. GitHub Pages（最推荐，适合开发者）

1. 在 GitHub 创建仓库，例如 `my-resume`
2. 将 `online-resume` 文件夹内所有文件上传
3. 进入仓库 **Settings → Pages**
4. Source 选择 `Deploy from a branch`，Branch 选 `main`，文件夹选 `/ (root)`
5. 保存后获得地址：`https://你的用户名.github.io/my-resume/`

**优点**：免费、稳定、可绑定自定义域名、展示 GitHub 主页链接

---

### 2. Gitee Pages（国内访问更快）

1. 在 [Gitee](https://gitee.com) 创建仓库并上传文件
2. 进入 **服务 → Gitee Pages**
3. 选择分支部署，获得 `https://用户名.gitee.io/仓库名`

**优点**：国内访问速度快，适合发给国内 HR

---

### 3. Vercel（一键部署，最简单）

1. 注册 [Vercel](https://vercel.com)
2. 导入 GitHub/Gitee 仓库，或拖拽 `online-resume` 文件夹
3. 自动获得 `https://项目名.vercel.app`

**优点**：部署最快，HTTPS 自动配置

---

### 4. Netlify

1. 注册 [Netlify](https://www.netlify.com)
2. 拖拽 `online-resume` 文件夹到 Deploy 区域
3. 获得 `https://随机名.netlify.app`

---

### 5. Cloudflare Pages

1. 注册 [Cloudflare](https://pages.cloudflare.com)
2. 连接 Git 仓库或直接上传
3. 免费 CDN，全球加速

---

## 使用建议

| 场景 | 推荐平台 |
|------|----------|
| 投递技术岗、放 GitHub 主页 | GitHub Pages |
| 发给国内公司 HR | Gitee Pages |
| 最快上线、不想折腾 | Vercel / Netlify |
| 需要自定义域名 | GitHub Pages / Cloudflare |

## 简历链接怎么写

- 求职邮件签名：`在线简历：https://你的地址`
- PDF 简历顶部：`Portfolio: https://你的地址`
- 微信/LinkedIn 个人简介中附上链接

## 导出 PDF

在浏览器打开简历后，点击左侧「打印 / 导出 PDF」，或按 `Ctrl+P` 选择「另存为 PDF」。
